# Project Status

Определи текущее состояние проекта.

Проверь:

1. Активные папки в `openspec/changes/`.
2. Текущий `change.md`.
3. Выполненные и невыполненные задачи.
4. Раздел текущего статуса в `docs/ROADMAP.md` или `PROJECT.md`.
5. Последний Git-коммит.
6. Наличие незакоммиченных изменений.

Ничего не изменяй.

Объясни простым языком:

- какой этап сейчас выполняется;
- что уже завершено;
- что осталось в текущей задаче;
- есть ли незакоммиченные изменения;
- какой следующий шаг указан в плане.
