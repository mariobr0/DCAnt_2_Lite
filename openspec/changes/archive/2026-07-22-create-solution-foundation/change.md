# Create Solution Foundation

## Зачем

Необходимо создать базовую архитектуру и каркас решения для проекта DCAnt 2.0. Это первый шаг разработки, который обеспечит правильное разделение ответственности между проектами и настроит общие параметры сборки перед написанием торговой логики.

## Требуемое поведение

- Создан solution файл `DCAnt2.slnx`.
- Созданы 4 production-проекта: `DCAnt2.Core`, `DCAnt2.Infrastructure`, `DCAnt2.Quantower`, `DCAnt2.Plugin`.
- Созданы 2 тестовых проекта: `DCAnt2.Tests`, `DCAnt2.QuantowerTests`.
- Задана целевая платформа .NET 10.
- Настроены зависимости между проектами согласно `ARCHITECTURE.md`.
- Добавлены общие файлы `.editorconfig` и `Directory.Build.props`.
- Проекты успешно собираются.

## Не входит

- Написание какой-либо доменной, бизнес- или торговой логики.
- Подключение SQLite или любых других внешних библиотек.
- Интеграция с API Quantower.
- Реализация каких-либо тестов.
